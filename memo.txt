implementing another transport layer for Erlang client

- implement a gen_server of the new transport that behaves thrift_transport, see thrift_socket_transport.erl for example
- implement a transport factory function that returns {ok, <module name that implements thrift_transport behaviour>}
- create a protocol factory by calling thrift_bynary_protocol:new_protocol_factory that accepts a transport factory function
- create a thrift client by calling thrift_client:start_link/2 that accepts a protocol factory and a service


implementing another transport layer for C++ server

- implement TTransport interface
- implement TTransportServer interface
- implement TTransportFactory interface
- create a server using TSimpleServer or TThreadPoolServer
